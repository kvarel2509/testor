Создать апи

1) Регистрация
2) Авторизация по токену
3) Список с тестами.
- get запрос дает список тестов
- post запрос позволяет создать тест
4) Страница с вопросами теста.
- get запрос показывает список вопросов и варианты ответов.
- post запрос позволяет добавить вопрос к тесту и варианты ответов к нему.
  Указывается, какие из них правильные
5) Апи прохождения теста. Выдает вопрос и варианты ответов. post запрос отправиляет фиксирует ответы
6) Статистика конкретного юзера по прохождениям тестов.
